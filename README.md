# vibecode starter
AIとの会話だけでアプリを開発する「バイブコーディング」専用スターターキットです。

### このキットを使うと
- **コードを一切書かずに** AIに何を作りたいか伝えるだけでブラウザで動くアプリが完成
- AIに「こういう機能を追加して」と伝えるだけで修正できる
- 作ったアプリはPCで実行し、ネットワーク経由で他の人も利用できるようになる

## こんな場面で活躍します
- 毎日同じ作業を繰り返しているが、プログラミングの知識がないため自動化できない
- アイデアはあるが、コードを書くのが面倒で実装できない
- コーディングの知識はあるが、細かい実装に時間を取られたくない

## 開発できるアプリの例
- 人事・総務: 社内アンケートの集計・ダッシュボード
- 経理・財務: 経費精算の自動化ツール
- 営業: 顧客管理ダッシュボード
- 工場・製造: 在庫チェックツール・生産スケジュール管理ツール
- カスタマーサポート: FAQ検索ツール・問い合わせ内容の自動振り分けツール

## 使用を開始する前に
- [Vibecode Starter セットアップ手順](https://docs.google.com/document/d/1h9XnBZfm4ZsFwhLhF9b02Hpk275S7VnF4Hpsi3zvd4w/edit?usp=sharing) を参考にPCに必要なソフトウェアをインストールしてください

## 開発の進め方
1. AIに作りたいアプリの内容を伝える
2. AIが指定された機能を実装し、アプリを起動
3. Google Chrome で http://localhost:7860 を開いて、アプリの動作を確認する
4. 修正したい点を AI に伝えると、AIが自動的にコードを修正、ブラウザの表示が自動更新される

コードを書く必要はありません。AIに指示するだけでアプリが完成します。

## ファイル構成

```
vibecode-starter/
├── app.py          # アプリ本体（AIがアプリの機能を実装するファイル）
├── pyproject.toml  # Pythonの追加ライブラリを管理するファイル
├── .env            # 環境変数（APIキーなど）
├── README.md       # このファイル
├── CLAUDE.md       # AIの振る舞いやルールを定義するファイル
└── agent-docs/     # アプリ開発を進める上でAIが参考にするドキュメント
    └── init/       
```
